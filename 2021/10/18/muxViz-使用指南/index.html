muxViz 使用指南

2021-10-18

安装

  1. 安装R-studio、R
  2. muxViz-3.1（无GUI）：https://github.com/manlius/muxViz
  3. muxViz-2.0（GUI）：https://github.com/wjj0301/Multiplex-Networks

输入数据格式

# config.txt

path_multilayer; path_to_layers_info; path_to_layers_layout
# 或者每层对应一个文件
path_layer_X; label_layer_X; path_to_layout_layer_X
...

# layers.txt
# 层信息

layerID layerLabel
1 L1
2 L2
3 L3
...

# .edges
# 层内连边关系，无headline
# 依次为 源节点ID、层ID、目标节点ID、层ID、边权值
1 1 53 1 3                                                                     
2 1 94 1 4
2 1 165 1 7
...

# 层间连边关系，对于multiplex，节点ID相同
1 1 1 2
2 1 2 2
...

# layout.txt
# 节点信息，可选，加入此文件后，edges文件nodeID可直接换成nodeLabel

nodeID nodeLabel
1 n1
2 n2
3 n3
...

绘图

library(muxViz)
library(igraph)
library(rgl)
set.seed(1)
# Network setup
Layers <- 3
Nodes <- 200
layerCouplingStrength <- 1
networkOfLayersType <- "categorical"
isDirected <- F
layer.colors <- brewer.pal(8, "Set2")
pathInfomap <- "infomap-0.x/Infomap"
nodeTensor <- list()
g.list <- list()
# 每层社团数
plantedGroupsPerLayer <- 4
# matrix of the stochastic block model
block.matrix <- matrix(0.1 / Nodes, plantedGroupsPerLayer,
                       plantedGroupsPerLayer)
diag(block.matrix) <- 2 * log(Nodes) / Nodes
# 总节点数平均分给四个社团
block.sizes <- rep(floor(Nodes / plantedGroupsPerLayer), plantedGroupsPerLayer)

for (l in 1:Layers) {
  #Generate the layers
  #双中括号提取修改元素
  g.list[[l]] <- sample_sbm(Nodes, pref.matrix=block.matrix,
                            block.sizes=block.sizes)
  
  #Get the list of adjacency matrices which build the multiplex
  nodeTensor[[l]] <- get.adjacency(g.list[[l]])
  #定义布局方式 "fr","drl", "auto", "kk", "comp", "dh"
  lay <- layoutMultiplex(g.list, layout="fr", ggplot.format=F, box=T)
  
  # Show the multiplex network
  plot_multiplex3D(g.list, layer.layout=lay, layer.colors=layer.colors,
                   layer.shift.x=0.5, layer.space=2,
                   layer.labels="auto", layer.labels.cex=1.5,
                   node.size.values="auto", node.size.scale=0.8,
                   show.aggregate=T)
  # 导出，有问题
  snapshot3d("../man/figures/multi_sbm.png", fmt = "png", width = 1024, height = 1024)
}

图像操作

GUI

linux下有问题，win下操作

  • multiplex 要画出层间连线只能用GUI，muxViz未提供接口，GUI源码仍然使用igraph，只能从sever.R中剥离此函数

  • GUI中给出不同节点或边指定颜色的接口，使用后闪退；可以按GUI的社团检测结果给出颜色

  • 多层社团检测使用后闪退
